type build_process = [ `Artifact of Ketrew_artifact.t
       | `Direct_command of Command.t
       | `Get_output of Command.t
       | `Long_running of string * string ]
Specification of how to build a target.
  • `Artifact a: literal, already-built, artifact,
  • `Direct_command c: a Command.t to run (should produce a Volume.t),
  • `Get_output c: a Command.t to run and get its stdout (should produce a value),
  • `Long_running (plugin_name, initial_run_parameters): Use a long-running plugin.
