val configure_ssh_batch_option : [ `Custom of string | `Dropbear | `Openssh ] -> unit
Configure global “Batch option”, (call ssh without password/question):
  • for OpenSSH, it is "-oBatchMode=yes",
  • for DropBear, it "-s".
